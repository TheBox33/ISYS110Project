# ISYS110Project
Sports Data
Manage sports data for multiple teams or sports.

The type of team, player and game data fill vary with the sports you choose!
This can be for college, high school or professional sports. 
This focuses on basic form processing.
This focuses on reading and writing to files. 
You will need to read and display the data sorted.
You will need to work with a collection such as a list, tuple or dictionary.  

Files
Player Information file stores the player information

Include 3 properties, such as the position or other property of your choosing (such as jersey number)
